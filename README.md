# Openapi Certifications and Partnerships

This repository is the public English-language index of Openapi SpA certifications, recognitions, memberships, registrations, patents, and certified partner ecosystem.

Openapi's public domain is [openapi.com](https://openapi.com). The primary public source for this repository is the official [Certifications, Special Recognitions and Memberships](https://openapi.com/certifications) page. Partner ecosystem information is sourced from the official [Search Openapi Partners](https://openapi.com/search-partner) page.

## Certifications

Openapi publishes the following certifications and authorizations:

- **UNI EN ISO 9001:2015**: quality management certification, certificate number 3482, issued by Istituto Giordano and Accredia.
- **UNI CEI ISO/IEC 25012:2014**: data quality certification, certificate number 023I-OPNP-25012.
- **ISO/IEC 27001:2022**: information security management certification.
- **UNI PdR 125:2022**: gender equality management system certification for API software design and production.
- **TULPS public security licence**: protocol number 397665/Area I TER O.S.P.
- **Postal operator service authorization**: licence number AUG/7280/2024.
- **Corporate messaging service authorization**.
- **Security clearance**: preventive authorization level "Highly Confidential" with NATO qualification SP-25479-AP.
- **SecurityMetrics PCI validation**: quarterly vulnerability assessment validation.
- **SOC 1 Type II and SOC 2 Type II certificates**: certifications related to the Google infrastructure used by Openapi's private cloud.
- **PCI-DSS Level 1**: certification related to Openapi's private cloud infrastructure.
- **Cloud Security Alliance STAR self-assessment**.
- **OpenStack Powered certification**.
- **Kubernetes Software Compliance**: compliance issued by the Cloud Native Computing Foundation (CNCF).
- **Code of Conduct on the use of Alphanumeric Aliases**.

## Special Recognitions

- **Software Made in Europe Label**: recognition that the software is designed and developed entirely in Europe.
- **Recognition of Solidarity Contributors**: acknowledgement issued by the Italian Ministry of Economy and Finance during the COVID-19 pandemic.
- **Legality Rating Score**: Openapi is listed by the Italian Competition Authority (AGCM) with a "Star ++" score.
